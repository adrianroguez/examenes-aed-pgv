# Exámenes Resueltos - AED y PGV

Este repositorio tiene como función principal almacenar y organizar exámenes resueltos y material de recuperación correspondiente a las asignaturas de:

- **AED** (Acceso a Datos)
- **PGV** (Programación de Servicios y Procesos)

Aquí encontrarás soluciones y ejemplos de código desarrollados para la preparación y estudio de estas materias.
